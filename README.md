# キラキラミュージックBOX

## 準備

1. ArduinoIDEをインストール
2. ArduinoIDEに最新のTimerOneとTimerThreeを入れる
3. Arduinoの回路を組む
4. ArduinoIDEを起動して、`mbox6.ino`を開く
5. Arduinoを起動して、`mbox6.ino`を書き込む

## 実行方法

Arduinoを起動する

|:--------------|:-----------|
| ArduinoIDE    | v1.6.7     |
